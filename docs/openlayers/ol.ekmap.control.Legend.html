Legend

Legend.

new ol.ekmap.control.Legend(options)

control/Legend.js, line 5
Name Type Description
options Object

Construction parameters.

Name Type Default Description
layers Array.<ol.Map>

List of layers for which you want to display legend.
target string

Specify a target if you want the control to be rendered outside of the map's viewport.
If target is equal to null or undefined, control will render by default.
title string LEGEND

Name of header.
tooltip string Legend

Tooltip of button.
className string

Style css of button.
showLabel boolean true

Show label of legend.
Example
var map = new ol.Map({https://server.ekgis.vn/ekmapserver/rest/services/145/MapServer
     //config....
 });
 var tiledMap = new ol.ekmap.TiledMapLayer({
      url: 'https://viegisserver.ekgis.vn/gserver/rest/services/35/MapServer'
 }).addTo(map);
 var legend = new ol.ekmap.control.Legend({
     layers: [tiledMap]
 });
 map.addControl(legend,"top-left");
