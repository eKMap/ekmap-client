Class: BasemapArcgisLayer

BasemapArcgisLayer

The BasemapArcgisLayer class use some arcgis layer and url of them from source.

new ol.ekmap.BasemapArcgisLayer(basemapType)

layer/BasemapArcgisLayer.js, line 4
Name Type Description
basemapType string

basemapType refers to the specific basemap you'd like to add.
Use one of "Streets", "Topographic", "Oceans", "OceansLabels", "NationalGeographic", "Physical", "Gray", "GrayLabels", "DarkGray", "DarkGrayLabels", "Imagery", "ImageryLabels", "ImageryTransportation", "ImageryClarity", "ImageryFirefly", ShadedRelief", "ShadedReliefLabels", "Terrain", "TerrainLabels" or "USATopo".
Example
var map = new ol.Map({
    container: 'map1',
    center: [103.9, 22.2],
    zoom: 6
});
var baseMap = new ol.ekmap.BasemapArcgisLayer('Topographic')
  .addTo(map);
