Symbol Circle configuration object.

Name Type Default Description
codomain Array.<number>

The data range allowed to be displayed in the chart is a one-dimensional array with a length of 2. The first element represents the lower limit of the value range, and the second element represents the upper limit of the value range.
maxR number

The maximum radius of the circle.
minR number

Minimum radius of circle.
fillColor string

The fill color of the circle, such as：fillColor: "#FFB980".
circleStyle Object

The basic style of the circle, this parameter controls the basic style of the circle, the priority is lower than circleStyleByFields and circleStyleByCodomain.
decimalNumber number

Data value array dataValues Decimal place of element value, data decimal place processing parameter, value range: [0, 16]. If this parameter is not set, the data will not be processed with decimal places when fetching the data value.
circleHoverStyle Object

The style of the circle hover state, valid when circleHoverAble is true.
circleHoverAble boolean true

Whether to allow the circle to use the hover state. At the same time, setting circleHoverAble and circleClickAble to false can directly shield the response of the graphics to thematic layer events.
circleClickAble boolean true

Whether to allow the circle to be clicked. At the same time, setting circleHoverAble and circleClickAble to false can directly shield the response of the graphics to thematic layer events.
showText boolean false

Additional text in the graphic, default: false.
textColor string

Text color. Default value: "#ffffff'"
textFont string

Additional text styles ["font-style", "font-variant", "font-weight", "font-size / line-height", "font-family"]. Example:'normal normal normal 12 arial,sans-serif'。
textPosition string

Additional text position. Settable value:"inside", "left", "right", top", "bottom", "end". Default value："inside"。
textAlign string

Horizontal alignment of additional text. Possible values: "start", "end", "left", "right", "center". By default, it is automatically set according to textPosition.
fontWeight string normal
