Class: WFS

mapboxgl.ekmap.WFS

Instantiates a WFS tile layer object given a base URL of the WFS service and a WFS parameters/options object..

new mapboxgl.ekmap.WFS(baseUrl, options)

Control options.

Name Type Default Description

Will use this token to authenticate all calls to the service.
