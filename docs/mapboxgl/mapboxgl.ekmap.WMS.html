Class: WMS

mapboxgl.ekmap.WMS

Instantiates a WMS tile layer object given a base URL of the WMS service and a WMS parameters/options object..

new mapboxgl.ekmap.WMS(baseUrl, options)

layer/WMS.js, line 4
Name Type Description
baseUrl string

a base URL of the WMS service.
options Object

Control options.

Name Type Default Description
token string

Will use this token to authenticate all calls to the service.
layers string

(required) Comma-separated list of WMS layers to show.
styles string

Comma-separated list of WMS styles.
tileSize string 256
bbox string

='{bbox-epsg-3857}'
format string 'image/jpeg'

WMS image format (use 'image/png' for layers with transparency).
transparent boolean false

If true, the WMS service will return images with transparency.
version string '1.1.1'

Version of the WMS service to use.
crs CRS null

Coordinate Reference System to use for the WMS requests, defaults to map CRS. Don't change this if you're not sure what it means.
uppercase boolean false

If true, WMS request parameter keys will be uppercase.
onClick boolean true

Fired when a pointing device (usually a mouse) is pressed and released at the same point on the map.
If false, not fired.
Example
var map = new mapboxgl.Map({
    container: 'divMapId',
    zoom: 3.9996619013972636,
    center: [-96.52559859275812, 39.21792880745451]
});
var wms = new mapboxgl.ekmap.WMS('https://demo.geo-solutions.it/geoserver/topp/wms', {
    layers: 'topp:states',
    transparent: true
}).addTo(map);
