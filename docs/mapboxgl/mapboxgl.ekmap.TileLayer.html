Construction parameters.

Name Type Default Description
urls Array.<string>

An array of one or more tile source URLs, as in the TileJSON spec.
url string

Required: URL of the Map Service with a tile cache.
token string

Will use this token to authenticate all calls to the service.
id string

Id of layer and source.
name string

Name of layer.
tileSize string 256

Units in pixels. The minimum visual size to display tiles for this layer. Only configurable for raster layers.
minzoom Number

The minimum zoom level for the layer. At zoom levels less than the minzoom, the layer will be hidden (between 0 and 24 inclusive).
maxzoom Number

The maximum zoom level for the layer. At zoom levels equal to or greater than the maxzoom, the layer will be hidden (between 0 and 24 inclusive).
visibility string visible

show or hide layer.
