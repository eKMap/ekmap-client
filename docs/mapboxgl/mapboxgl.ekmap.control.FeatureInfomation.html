Name Type Description
options Object

Construction parameters.

Name Type Default Description
icon string fa-flickr

Icon of button.
showButton Boolean true

If showButton = false, button control is not displayed.
setStyle Boolean true

If setStyle = false, the selected feature will not set style and vice versa it will set style default.
target string

Specify a target if you want the control to be rendered outside of the map's viewport.
If target is equal to null or undefined, control will render by default.
