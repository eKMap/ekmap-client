new mapboxgl.ekmap.control.Draw(options, displayControlsDefault, modes)

control/Draw.js, line 4
Name Type Default Description
options Object

Construction parameters.

Name Type Description
controls Object

Hide or show individual controls. Each property's name is a control, and value is a boolean indicating whether the
control is on or off. Available control names are point, line_string, polygon, trash, combine_features and uncombine_features. By
default, all controls are on. To change that default, use displayControlsDefault..
displayControlsDefault Boolean true

The default value for controls. For example, if you would like all controls to be off by
default, and specify an allowed list with controls, use displayControlsDefault: false.
modes Object

Over ride the default modes with your own.
